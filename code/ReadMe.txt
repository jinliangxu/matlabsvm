-- Simple Guide --
1. Prepare data set and OSU-SVM lib
2. Copy all program scripts to OSU-SVM lib folder
3. Update loadDataAndTest.m and set correct path to your training data set
	e.g.load('E:\proj\matlabsvm\data\iris_class.mat')
4. Simply type loadDataAndTest command to run testing
5. Collect results
